# Roadmap

This roadmap describes the intended direction of the project. Priorities may change based on testing, controller compatibility and user feedback.

## v1.0 — Stable foundation

- Local monitoring for tested Datakom D500 MK2 and D502 controllers
- Live physical LCD display
- Remote LCD menu navigation
- HACS installation
- Home Assistant diagnostics
- Bundled Lovelace cards
- English and Slovenian user interface
- Digital output monitoring

## v1.1 — Remote operation and history

- Remote RUN, STOP, AUTO and TEST controls
- Explicit confirmations and safeguards for operating-mode commands
- Alarm history
- Event log

## v1.2 — Broader monitoring

- Read-only controller configuration
- Digital input monitoring
- Additional diagnostic entities
- Improved service information
- Testing and compatibility work for additional Datakom D-Series models

## Future ideas

- Better model-specific output naming
- More dashboard examples
- Protocol documentation and test fixtures
- Expanded automated tests with a simulated controller
