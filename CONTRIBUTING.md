# Contributing

Contributions, testing reports and protocol findings are welcome.

## Development

1. Fork the repository.
2. Create a feature branch.
3. Keep the integration read-only unless a change is explicitly discussed first.
4. Run the HACS and Hassfest validation workflows.
5. Open a pull request with a clear description and test details.

## Device reports

When reporting compatibility, include:

- Controller model
- Hardware revision
- Firmware version
- Which entities work or fail
- Relevant Home Assistant logs with private network details removed

Do not publish passwords, cloud credentials, private IP addresses or full packet captures containing sensitive data.
